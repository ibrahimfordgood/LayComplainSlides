# ComplainBot Pitch Deck

This is a static HTML pitch deck for ComplainBot.

## Hosting

This project is a simple static site and can be hosted on any static site provider.

### Vercel
1. Install Vercel CLI: `npm i -g vercel`
2. Run `vercel` in this directory.
3. Follow the prompts.

### Netlify
1. Drag and drop this folder into the Netlify Drop dashboard.
2. Or use the Netlify CLI: `netlify deploy`

### GitHub Pages
1. Push this repository to GitHub.
2. Go to Settings > Pages.
3. Select the branch and folder (root) to deploy.

## Local Development
Simply open `index.html` in your browser.
